private subroutine ExtDataExtractIntegers(string, iSize, iValues, delimiter, verbose, rc)

Extract integers from a character-delimited string, for example, “-1,45,256,7,10”. In the context of Chem_Util, this is provided for determining the numerically indexed regions over which an emission might be applied.
