MODULE: MAPL_OrbGridCompMod

Author: GMAO SI-Team

The module MAPL_OrbGridCompMod is an ESMF gridded component implementing the MAPL-5 Nominal Orbit Calculator.

It was developed for MAPL-5 release Fortuna 2.3 and later.
