MODULE: MAPL_ConfigMod

Author: GMAO SI-Team

MAPL_ConfigMod implements extensions that allow extending an ESMF_Config object. Otherwise, ESMF only provides a constructor that loads the data from a text file.


Uses

    • ESMF
